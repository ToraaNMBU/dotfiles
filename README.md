# dotfiles


## Requirements

### Git

```
sudo apt install git -y
```

### Stow
```
sudo apt install stow -y
```

## Installation

Checkout this branch 


```
git checkout git@github.com:ToraaNMBU/dotfiles.git -b ubuntu22.04
cd dotfiles
```

Then execute the GNU stow to enable the symlinks
```
stow . 
```
